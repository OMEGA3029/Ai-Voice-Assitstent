# Ai-Voice-Assitstent
This project aims to develop a personalized AI voice assistant named Luffy, capable of understanding voice  commands, executing system-level tasks, playing YouTube videos, searching the web, sending emails, and generating responses using the Groq LLM API—all while interacting naturally using text-to-  speech and speech recognition technologies. 

#Features

Voice-controlled commands — Speak naturally to perform actions

YouTube integration — Opens YouTube and plays the first matching video automatically

Email support — Send emails to saved contacts using Gmail SMTP

Web search — Search directly on Google using voice

App control — Open apps like Notepad, Chrome, Gmail

Groq AI integration — Intelligent responses using the Groq API (LLaMA 3.3 model)

System control — Get time, restart, or shutdown the system by voice

Offline speech-to-text — Uses speech_recognition and pyttsx3 for real-time interaction
